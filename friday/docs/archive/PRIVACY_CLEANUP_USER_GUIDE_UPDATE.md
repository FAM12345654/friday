# Privacy Cleanup User Guide Update

## Ziel

Dieses Dokument beschreibt die Aktualisierung der Nutzer-Doku fuer guarded Privacy Cleanup Writes.

## Nutzerhinweis

Die read-only Preview bleibt auf Menuepunkt `7`.

Der guarded Cleanup-Write ist auf Menuepunkt `8` und braucht:

- konkrete Zielpfad-Eingabe,
- Safety Smoke `PASS`,
- Guard-Freigabe,
- exakten harten Token.

Rueckkehr ist jetzt Menuepunkt `9`.

## Safety-Hinweis

Nur Datei-Cleanup fuer Exporte, Backups und Restore-Kopien ist per CLI vorgesehen.

Kontakt-Kontexte und Review-Historie bleiben nicht per CLI-Cleanup freigegeben.
