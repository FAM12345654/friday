# Friday Mobile Task Forward Field

## Ziel

Friday Mobile hat im Aufgabenbereich ein neues Feld:

`Weiterleiten an Kollege`

Damit kann beim Erstellen einer Aufgabe ein Kollege als lokaler Hinweis eingetragen werden.

## Verhalten

Wenn ein Name eingetragen wird, speichert Friday die Aufgabe mit einer Notiz:

`Weiterleiten an: <Name>`

Die Notiz wird in der Aufgabenkarte angezeigt.

## Safety

- Es wird keine Nachricht gesendet.
- Es wird keine E-Mail gesendet.
- Es wird kein WhatsApp/SMS-Versand ausgefuehrt.
- Es wird keine externe Integration genutzt.
- Das Feld ist nur eine lokale Markierung an der Aufgabe.

## Naechster sinnvoller Schritt

Ein spaeteres Gate kann daraus einen echten Draft-/Review-Flow machen:

- Aufgabe fuer Kollege vorbereiten,
- Nachricht als Entwurf anzeigen,
- Versand weiterhin nur nach explizitem separatem Gate.
